Функциональные ряды.

$13. Понятие функциональной последовательности и её сходимости.
Рассмотрим последовательность функций:
f₁(x), f₂(x), ..., fn(x), определённую на интервале (a;b).
Пусть для каждого значения x∈(a;b) эта последовательность имеет конечный предел.
lim fn(x) = f(x)
n→∞

> Функциональная последовательность {fn(x)}, определённая на интервале (a;b)
называется сходящейся в этом промежутке к предельной функции f(x), если
∀ε>0 ∃N(ε,x) : ∀n ≥ N → |fn(x) - f(x)| < ε

=Теорема (критерий Коши для функциональной последовательности):
Для того, чтобы последовательность {fn(x)} функций, определённых на интервале (a;b)
сходилась к функции f(x), необходимо и достаточно, чтобы
∀ε > 0 ∃N(ε,x) : ∀n,m∈N → |fn+m(x) - fn(x)| < ε


$14. Понятие функционального ряда и его сходимости.
Рассмотрим последовательность функций u₁(x), u₂(x), ..., un(x), определённых на
интервале (a;b).

> Выражение вида:             ∞
u₁(x) + u₂(x) + ... + un(x) = Σ un(x) (1) называется "функциональным рядом".
                             n=1
Пусть x₀∈(a;b), для этого значения x из данного ряда (1) получим числовой ряд
 ∞
 Σ un(x₀) (2)
n=1
Если ряд (2) сходится, то говорят, что функция ряда (1) сходится в точке x₀∈(a;b).

Обозначим сумму ряда (2) S(x₀)
Если функциональный ряд (1) сходится в каждой точке x∈(a;b), то он называется
"сходящимся в интервале (a;b)".
В этом случае каждой точке x∈(a;b) будет отвечать определённое число, являющееся
суммой соответствующего числового ряда.

Таким образом, функциональный ряд будет определять некоторую функцию S(x)
в интервале (a;b), которую называют "суммой функционального ряда (1)".

Промежуток, в котором ряд (1) сходится, называют "областью сходимости ряда".

Как и в случае числовых рядов можно образовать конечные суммы ряда (1)
Sn(x) = u₁(x) + u₂(x) + ... + un(x)
которые будем называть "частичными суммами ряда (1)".

> Ряд (1), члены которого определены в интервале (a;b) называют сходящимся в этом
промежутке к функции S(x), если
∀ε>0 ∃N(ε,x) : ∀n≥N → |Sn(x) - S(x)| < ε
lim Sn(x) = S(x)
n→∞

Т.к. между сходимостью функциональной последовательности и функционального ряда
имеется тесная связь, то для функциональных рядов можно сформулировать критерий Коши,
аналогичный критерию Коши сходимости функциональной последовательностью.

=Теорема (критерий Коши для функционального ряда):
Для того, чтобы функциональный ряд (1), определённый на интервале (a;b) сходился к
сумме S(x), необходимо и достаточно, чтобы
∀ε > 0 ∃N(ε, x) : ∀n≥N, ∀m∈N → |un₊₁(x) + un₊₂(x) + ... + un+m(x)| < ε


$15. Равномерная сходимость функционального ряда.
Из определения сходимости функционального ряда следует, что натуральное число N,
обеспечивающее выполнение основного неравенства |Sn(x) - S(x)| < ε зависит как от ε,
так и от x∈(a;b). Если зависимость от переменной x отсутствует, то говорят о
"равномерной сходимости функционального ряда".
      ∞
> Ряд Σ un(x) называется "равномерно сходящимся" к сумме S(x), если
     n=1
∀ε>0 ∃N(ε) : ∀n≥N → |Sn(x) - S(x)| < ε

=Теорема:
Для того, чтобы ряд был равномерно сходящимся в интервале (a;b) необходимо и
достаточно, чтобы
∀ε > 0 ∃N(ε, x) : ∀n≥N, ∀m∈N → |un₊₁(x) + un₊₂(x) + ... + un+m(x)| < ε

=Доказательство:
1. Необходимость.
Т.к. функциональный ряд равномерно сходится в интервале (a;b) к сумме S(x), то
∀ε>0 ∃N(ε) : ∀n≥N, ∀x∈(a;b) → |Sn(x) - S(x)| < ε/2
с другой стороны, на том же основании, выполняется неравенство
|Sn+m(x) - S(x)| < ε/2 ∀m∈N

Отсюда имеем |Sn+m(x) - Sn(x)| = |Sn+m(x) - S(x) + S(x) - Sn(x)| ≤
≤ |Sn+m(x) - S(x)| + |Sn(x) - S(x)| < ε/2 + ε/2 = ε

2. Достаточность.
Пусть ∀x∈(a;b) выполняется неравенство |Sn+m(x) - Sn(x)| < ε
тогда на основании критерия Коши, данный ряд сходится на интервале (a;b) к некоторой
сумме S(x), т.е.
lim Sk(x) = S(x)
k→∞
Тогда, устремляя n в бесконечность (что влечёт за собой стремление m+n→∞) получим
|S(x) - Sn(x)| < ε
Это неравенство означает, что данный ряд равномерно сходится.
✓ Теорема доказана.
