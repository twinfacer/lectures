I. Приложения определённого интеграла.
$1. Понятие квадрируемой фигуры и её площади. Основные свойства квадрируемых фигур.

> Произвольное ограниченное множество точек плоскости будем называть "плоской фигурой".
Если плоскую фигуру можно представить как объединение конечного числа непересекающихся
прямоугольников, то такую фигуру назовём "клеточной".

> Под "прямоугольником" будем понимать множество точек K = { (x,y): a1 ≤ x ≤ b1, a2 ≤ x ≤ b2 }
! В указанном определении двойные неравенства могут быть и строгими.

> "Площадью прямоугольника" К назовём число (b1 - a1)*(b2 - a2), независимо от того,
принадлежат или не принадлежат множеству K его граничные точки.

> "Площадью клеточной фигуры" назовём сумму пощадей прямоугольников, её составляющих.

Можно показать, что площадь клеточной фигуры независит от способа разбиения её на
прямоугольники, а так же, что площадь клеточной фигуры не отрицательна и обладает
следующими свойствами:
1) Аддитивность (площадь объединения двух непересекающихся клеточных фигур равна сумме
их площадей)
2) Инвариантность (площади двух равных клеточных клеточных фигур одинаковы)
3) Монотонность (если клеточные фигуры F1 и F2 таковы, что F1 c F2, то площадь фигуры F1
не превосходит площадь фигуры F2)

> Плоскую фигуру F назовём "квадрируемой", если ∀ε>0 ∃ q, Q: q c F c Q (1)
и выполняется неравенство:
0 ≤ S(Q) - S(q) < ε (2)

Пусть плоская фигура F квадрируема, тогда площадью этой фигуры назовём число S(q) ≤ S(F) ≤ S(Q) (3)
Для любых клеточных фигур q и Q удовлетворяющих условию (1)

=Теорема:
Для любой квадрируемой фигуры F число S(F) существует и единственно, причём:
S(F) = sup S(q) = inf S(Q) (4)


$2. Необходимое и достаточное условие квадрируемости.
=Теорема:
Для того, чтобы плоская фигура F была квадрируемой необходимо и достаточно,чтобы для
любого ε>0 существовали такие квадрируемые плоские фигуры ~q и ~Q, что
~q c F c ~Q, 0 ≤ S(~Q) - S(~q) < ε, (1)
где S(~q) и S(~Q) - площади фигур ~q и ~Q соответственно.

=Доказательство:
+Необходимость условий (1) очевидно, т.к. по определению квадрируемой фигуры эти условия
выполняются, если ~q = q, а ~Q = Q, где q и Q - клеточные фигуры, удовлетворяющие
условиям из определения квадрируемой фигуры.

+Достаточность
Фиксируя произвольное число ε>0 найдём в силу условий (1) такие квадрируемые плоские
фигуры ~q и ~Q, что ~q < F < ~Q и выполняется неравенство 0 ≤ S(~Q) - S(~q) < ε/2  (2)
Т.к. ~q и ~q - квадрируемые плоские фигуры, то существуют клеточные фигуры q' и Q', такие что
q' c ~q, ~Q c Q', 0 ≤ S(~q) - S(Q') < ε/4, 0 ≤ S(Q') - S(~Q) < ε/4  (2)
Из условий (2) и (3) q' c F c Q' и 0 ≤ S(Q') - S(q') < ε
Это означает, по определению, что F - квадрируемая фигура, причём S(F) = inf S(~Q) = sup S(~q)

! Можно доказать, что площадь квадрируемой фигуры обладает свойствами аддитивности,
инвариантности и монотонности.


$3. Вычисление площади в декартовых координатах.

нахождении площади криволинейной трапеции. Т.е. фигура F, задаваемая на плоскости (x,y)

F = { (x,y): a ≤ x ≤ b, 0 ≤ y ≤ f(x) }, где f(x) - функция, непрерывная на отрезке (a,b)

=Теорема:
Криволинейная трапеция F - квадрируемая фигура, площадь которой S выражается формулой:
    b
S = ∫f(x)dx (1)
    a

В параграфе "О введении понятия определённого интеграла".

    b

    a

$4. Вычисление площади в полярных координатах.
Пусть кривая Г задана в полярной системе координат уравнением:
ρ = ρ(φ), α ≤ φ ≤ β
Где ρ(φ) - не отрицательная и непрерывная на отрезке [α, β] функция.
Тогда плоскую фигуру G, ограниченную кривой Г и быть может отрезками двух лучей,
составляющих с полярной осью углы α и β называют "криволинейным сектором".

=Теорема:
Криволинейный сектор есть квадрируемая фигура, площадь которой вычисляется по формуле:
        β
S = 1/2 ∫ρ^2(φ)dφ       (1)
        α

=Доказательства
Пусть T = { φi, i = 0..n } - разбиение отрезка [α,β].
mi, Mi - наименьшее и наибольшее значения функции ρ(φ) на отрезке Δi=[φi-1, φi], i=1..n
Обозначим через qi и Qi круговые секторы, ограниченные лучами φ=φi-1 и φ=φ-i и дугами
окружностей, радиусов mi и Mi соответственно.
Если q - объединение круговых секторов q1, q2, ... qn,
а Q - объединение секторов Q1, Q2, ... Qn, то q c G c Q

Т.к. круговые секторы являются квадрируемыми фигурами, то фигуры q и Q так же квадрируемы,
а их площади соответственно равны.
           n
S(q) = 1/2 Σ mi^2 - Δφi
          i=1

           n
S(Q) = 1/2 Σ Mi^2 - Δφi
          i=1

Отсюда следует, что S(q) и S(Q) совпадают соответственно с нижней и верхней суммами Дарбу
для функции 1/2ρ^2(φ) на отрезке [α, β].
                                  β
Поэтому sup S(q) = inf S(Q) = 1/2 ∫ρ^2(φ)dφ
                                  α
С другой стороны sup S(q) = inf S(Q) = S(G).
Таким образом, G - квадрируемая фигура и справедлива формула (1).
