$1. Определение и свойства сходящихся рядов.

> Выражение a₁ + a₂ + ... + an + ..., где {an} - заданная числовая последовательность
называется "числовым рядом" и обозначается
∞

n=1

     n

    k=1
      ∞

     n=1

                              n→∞               ∞

                                               n=1

$2. Критерий Коши сходимости ряда.

=Теорема:           ∞
Для сходимости ряда Σ an (1), необходимо и достаточно, чтобы
                   n=1
∀ε>0 ∃Nε : ∀n ≥Nε, ∀p∈N → | an+1 + an+2 + ... + an+p | < ε               (2)

=Доказательство:
an+1 + an+2 + ... + an+p = Sn+p - Sn, где Sn - n-ая частичная сумма ряда (1), то
условие (2) означает, что {Sn} (последовательность) частичных сумм ряда (1) является
фундаментальной.
В силу критерия Коши для сходимости числовой последовательности, условие (2)
равносильно существованию конечного предела последовательности Sn, т.е. равносильна
сходимости ряда (1).
✓ Что и требовалось доказать.

! Если условие (2) не выполняется, т.е. ∃ε₀ > 0 : ∀k∈N ∃n≥k, ∃p∈N :
|an+1 + an+2 + ... + an+p| ≥ ε₀          (3)
То ряд (1) расходится
                                  ∞
# Доказать, что гармонический ряд Σ 1/n расходится
                                 n=1
∀k∈N n=k, p=k, тогда
1 / (n+1) + 1 / (n+2) + ... + 1 / (n+p) = 1 / (k+1) + 1 / (k+2) + ... + 1 / 2k = 1/2
ε₀ = 1/2
Таким образом, имеет место условие (3), а значит гармонический ряд расходится.


$3. Критерий сходимости ряда с неотрицательными членами.

=Теорема:       ∞
Если члены ряда Σ an (1) неотрицательны, т.е. ∀n ∈ N  an ≥ 0, то для сходимости этого
               n=1
ряда необходимо и достаточно, чтобы последовательность его частичных сумм {Sn} была
ограничена сверху, т.е.
                   n
∃M>0 : ∀n∈N → Sn = Σ ak ≤ M         (2)
                  k=1

=Доказательство:
Последовательность Sn - возрастающая, т.к. разность Sn - Sn-1 = an ≥ 0.
Если ряд (1) с неотрицательными членами сходится, т.е.
∃ lim Sn = S < ∞, то по теореме "о пределе возрастающей последовательности"
  n→∞
∀n∈N  Sn ≤ S, т.е. выполняется условие (2).
Обратно, если ряд (1) с неотрицательными членами удовлетворяет условию (2), то
возрастающая последовательность Sn ограничено сверху.
Следовательно она имеет конечный предел, т.е. ряд (1) сходится.
✓ Что и требовалось доказать.
